** Install TypeScript

*** HOW

#+begin_src shell
  sudo npm install -g typescript
  # Check the TypeScript Version
  tsc -v
#+end_src

** Hello World

1) Create the file HelloWorld.ts

   #+begin_src js
     let str: string = 'Hello World';
     console.log(str);
   #+end_src

2) Compile the ts file

   #+begin_src shell
     tsc HelloWorld.ts
   #+end_src
   
3) execute the js file

   #+begin_src shell
     node HelloWorld
   #+end_src
